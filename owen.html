HOTEL TH

Hotel TH is the project that we've made on my 2nd year as a IT student in the CVSU Bacoor. The system is coded in C# windows form. The application is all about hotel booking system, it has two user authentication, the client and the client can book, see its bookings, cancel booking, update the profile. while the admin can, accept bookings, edit the pictures, prices make employee account, make client account and view bookings









ST CINEMA

This project was done on my 1st year as an IT student. This website is all about online cinema bookings. The system have two side, the client and the admin. The client can view the now showing, incoming movies. Client can also make and cancel bookings. The admin side on the other hand can update the movies in the website, accept / decline bookings and have its dashboard
